Certificate Profile

Represents an exercise certificate that proves a group has been able to communicate successfully with the system.

Attributes

  • certificate: generated certificate (string)
  • generated: timestamp for when the certificate was generated (string)
  • group: handle of the group the certificate was generated for (string)

Certificates Profile

Profile for the certificate control resource that includes instructions on how to request a certificate and get notified when it is ready, as well as where it was placed.

Attributes

  • exchange: name of the exchange that will be used for notifications (string)

Connection Template Attributes

  • host: RabbitMQ server address (string)
  • port: RabbitMQ server port (integer)
  • virtual_host: Virtual host placeholder (string)

Group Profile

Represents a group known to the system.

Attributes

  • handle: unique handle for the group (string)
  • name: full name of the group (string)
