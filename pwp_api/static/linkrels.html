pwpex:certificates

Leads to a resource that is used for requesting certificates from the backend. Only accessible by the group's API key.

pwpex:create-certificate

Describes how to make a generated certificate available for download. Only accessible with worker API key.

pwpex:get-certificate

Link to a generated certificate. Only accessible by the group's API key.

pwpex:groups

Leads into the group collection resource which lists all public information for all registered groups.

pwpex:notify-listen

Contains a URI template to connect to RabbitMQ virtual host with your group's credentials. Also contains a template that can be filled to form the connection in cases where the URI cannot be used directly.

pwpex:request-certificate

Contains instructions on how to apply for an exercise certicate for your group. Only accessible by the group's API key.
